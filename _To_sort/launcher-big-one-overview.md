## The Launcher

**THE BIG ONE:** if in overview you can _select_ views, and views can _provide context_ (file navigator → current selection or folder; text viewer → current file), then the launcher can be evoked _against that context_. You navigate somewhere, select files, zoom out, maybe also pull in a couple script viewers, invoke the launcher — out pops a new agent session with context already bound. This is the AI integration point.

Calculator mode ergonomics: `Esc` = dismiss, `Ctrl+C` = copy result, `Enter` = pop the calculator out and retain the expression (more may be coming).
