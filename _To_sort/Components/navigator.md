# The Navigator

> Find the things to attend to.

Miller-column file browser. In Kelp, the floor of every [[strip]] — always present, not a window. Approximated now by Thunar.

Two modes of use:
- **Navigate** — fast, gestural, go-to for known paths.
- **Forage** — spatial, discovery-oriented browsing.

## Actions

- **Lift** a file → into the [[Purse]], held for examination.
- **`Ctrl+Enter` on a folder** → new sibling [[strip]] rooted there.
- **Select files + invoke [[launcher]]** → routes or acts on the selection.

---

## Ideas

**Gesture navigation** — entering navigate mode teleports cursor to the left screen edge. A directional overlay (like macOS dock magnification) indicates which subfolder you're steering toward. Threading through hierarchy by pointing, not clicking. Fast for familiar paths.

**Content preview on floor** — files below a size threshold render content directly on the floor as spatial objects. Small markdown notes, config snippets, quick captures: legible at a glance without lifting. Short-circuits [[Purse]] preview for text.
